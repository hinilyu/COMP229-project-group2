# Grocery Store Inventory System

This is a COMP229 Project by Group 2

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory and install dependencies

```bash
  cd client
  npm install
  cd ../Product
  npm install
```

Start the server in client folder

```bash
  npm start
```

In client folder

```bash
  ng serve --port 8081
```
